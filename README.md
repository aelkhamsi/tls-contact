# TLS Contact
                                                                                                                  
TLS Contact Web Scrapper

## Usage

1. Create an .env file and populate it with the following data:
   - TLS_EMAIL
   - TLS_PASSWORD
   - TLS_URL
   - SMTP_USER
   - SMTP_PASSWORD

2. Install **npm** and **node**, and run the following commands:
``` 
npm i
npm run start
```
